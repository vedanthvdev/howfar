Welcome to HowFar

HowFar adds walking, driving, and cycling times next to every address it spots on the web. It uses Google Maps under the hood, calling Google directly from your browser with a key you control.

  • Your key stays on this device — we don’t have a server.
  • Google gives you $200 / month of free credit.
  • Setup takes about 5 minutes.

Get a Google Maps API key

You’ll need a key from Google Cloud. It’s free for the first $200 of usage every month — far more than HowFar will ever spend on personal browsing.

  1. Open Google Cloud Console ↗

    Sign in (or create a Google account).

  2. Create a project.

    Anything you call it works. Free tier — no credit card needed for the first 90 days.

  3. Enable these two APIs:
    • Geocoding API
    • Directions API

    Search for each by name in the API Library, then click Enable.

  4. Create the key.

    APIs & Services → Credentials → Create Credentials → API key. Copy the value Google shows you.

  5. Restrict the key (recommended).

    Click Edit API key. Under API restrictions, choose Restrict key and select Geocoding API and Directions API. That’s it — leave application restrictions on None.

Paste your key

We’ll do a one-call test against Google to confirm it works before saving anything.
